# Unlimited Shortcuts 🚀

Chromeの新しいタブページで**無制限のショートカット**を管理できる拡張機能です。

デフォルトのChromeでは10個までしかショートカットを登録できませんが、この拡張機能を使えば好きなだけショートカットを追加できます！

![Screenshot](screenshot.png)

## ✨ 機能

### ショートカット管理
- **無制限のショートカット登録** - 10個の制限なし！
- **ドラッグ&ドロップで並び替え**
- **右クリックで編集・削除**
- **Faviconの自動取得** - サイトのアイコンを自動表示
- **カスタムアイコン設定** - 好きな画像URLを指定可能

### グループ機能
- **グループ分け** - 「仕事」「AI」など自由に分類
- **グループの並び替え** - ドラッグ&ドロップ
- **15色からカラー選択**
- **ショートカットをグループ間移動**

### その他
- **時計・日付表示** - 右上にコンパクト表示
- **Google検索バー**
- **ダークテーマ** - 目に優しいデザイン
- **コンパクトなUI** - 多くのショートカットを一度に表示

## 📥 インストール方法

### 方法1: GitHubからダウンロード

1. このリポジトリをダウンロード
   - 緑色の「**Code**」ボタン → 「**Download ZIP**」をクリック
   - または `git clone https://github.com/YOUR_USERNAME/chrome-shortcuts-extension.git`

2. ZIPファイルを解凍

3. Chromeで `chrome://extensions/` を開く

4. 右上の「**デベロッパーモード**」をオンにする

5. 「**パッケージ化されていない拡張機能を読み込む**」をクリック

6. ダウンロードしたフォルダを選択

7. 完了！新しいタブを開くとカスタムページが表示されます 🎉

## 📖 使い方

| 操作 | 方法 |
|------|------|
| ショートカット追加 | 右上の「**+**」ボタン |
| グループ追加 | 右上の「📁+」ボタン |
| 編集・削除 | ショートカットを**右クリック** |
| 並び替え | **ドラッグ&ドロップ** |
| グループ移動 | ショートカットを別グループに**ドラッグ** |
| グループ編集 | グループ横の「**⋮**」をクリック |

### カスタムアイコンの設定

1. ショートカットの追加/編集画面を開く
2. 「カスタムアイコン」欄に画像URLを入力
3. プレビューで確認して保存

※空欄の場合はサイトのfaviconが自動取得されます

## 🛠️ 技術仕様

- **Manifest Version**: V3
- **ストレージ**: chrome.storage.local
- **フレームワーク**: なし（Vanilla JS）
- **フォント**: Noto Sans JP

## 📁 ファイル構成

```
chrome-shortcuts-extension/
├── manifest.json      # 拡張機能の設定
├── newtab.html        # 新しいタブページ
├── styles.css         # スタイルシート
├── script.js          # メインのJavaScript
├── icons/             # 拡張機能のアイコン（任意）
└── README.md          # このファイル
```

## ⚠️ 注意事項

- この拡張機能をインストールすると、Chromeのデフォルトの新しいタブページが置き換えられます
- デフォルトのショートカットは引き継がれないため、再度登録が必要です
